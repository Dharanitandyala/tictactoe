Tic-Tac-Toe Web Application

This project is a simple web-based Tic-Tac-Toe game developed using HTML, CSS, and JavaScript. It allows two players to enter their names, take turns marking the board, and automatically detects wins and draws.

Project Overview

The goal of this project was to understand the basics of web development, including layout design, DOM manipulation, event handling, and user interaction. The application includes a responsive design and a reset option to restart the game without refreshing the page.

Features

Two-player gameplay

Name input functionality

Win and draw detection

Reset option

Responsive and clean UI
